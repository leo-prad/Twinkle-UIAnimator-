# Visual effects

A grab-bag of decorative effects. Most are **automatic** (driven by tags/attributes);
`Shake` is a scripted method.

| Effect | Activation | Docs |
|--------|------------|------|
| EasyVisuals special effects | Tag `SpecialEffects` + `EffectType` attribute | [special-effects.md](special-effects.md) |
| Continuous rotation | Tag `SpinUI` (elements) / `SpinGradient` (gradients) | [spinning.md](spinning.md) |
| Click ripple | Module script attributes (`RippleEnabled`, …) | [ripple.md](ripple.md) |
| Shake / vibration | `UIAnimator.Shake(...)` | [shake.md](shake.md) |
